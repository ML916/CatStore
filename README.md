# CatStore
Kodtest för Sogeti

Mobilapplikation skapad med Xamarin som agerar frontend mot kattaffärs-API. Applikationen består av en detaljmeny med tre flikar, en för att bläddra i sortimentet, en för att granska varukorg och slutföra köp samt en för att granska status på utförda beställningar hämtning av kvitton.
